The Celestial Wayfinder

A DEF CON badge built for on-grid and off-grid hackability. It has an ESP32-S3 MCU, 915MHz LoRa radio, GPS, digital LED compass, mechanical keyboard switches, and a tactile encoder. Rounding out the UI is a 128x128 pixel OLED screen, a haptic feedback motor, and a miniature magnetic buzzer.

The badge is has open-source firmware for your hacking pleasure, an electronic module system using card-edge connectors, and a standard for cosmetic customization so you can tune the device to fit your aesthetic and environment!

DEF CON Pickup

Find each other.
Off the grid.

The Celestial Wayfinder broadcasts your GPS coordinates over LoRa radio when you send a message — no external infrastructure needed. Other Wayfinders in range receive your position and their LED compass ring points at the source of your message.

  • Radio: 915MHz LoRa mesh radio — self-healing, zero config, no infrastructure
  • Microcontroller: ESP32-S3 Microcontroller with 16MB of Flash
  • Payload: GPS location + name + message payload, broadcast to every badge in range
  • Wayfinding: LED compass ring points directly toward source of a received message
  • Display: 128x128 pixel, 1.5" OLED display. Retro, but satisfying.
  • User Interface: easy 4-button UI can be learned in seconds
  • Tactile: 5× mechanical keyboard switches, haptic motor, magnetic buzzer, and encoder with detents
  • Portable: rechargable lithium-ion battery, ~1hr charge time
  • Power Input: USB-C, or auxilary 4-20VDC input with optional barrel jack
